必要なライブラリ
openpyxl
tkinter

【概要】
ログ内にあるアカウントIDの抽出を主な目的としたスクリプトです。
正規表現を使うことにより好きなパターンで抽出します。
抽出した文字列はエクセルファイルへ出力され、文字列は重複されません。

【補足】
オプションに -s もしくは --select を追加して実行することにより
ファイル選択のダイアログが出力されます。
上記オプションがない場合はconfig.iniに書かれているログを読み込みます。
pythonが実行できる環境であればOSは問わないはずです。

config.iniでregexセクション内のvarディレクティブで
文字列を抽出するための正規表現を指定します。

【ファイル内容】
extractLog.py ... 本体ファイル
config.ini    ... 設定ファイル
extract.bat   ... windows用バッチファイルでファイル選択ダイアログを出力し実行
sample.log    ... 文字列の抽出を試せるサンプル用のログ
