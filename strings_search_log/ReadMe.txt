Program name 【Strings search in the log】
Made 2023/07/28
©EngineerTN

【 概要 】
検索文字を指定（複数可で、且つテキストorCSVで指定可※）し、
対象のログから何行目に指定文字がマッチするかを判定します。
※検索文字をテキスト or CSVファイルで指定する場合は、改行区切りで複数指定出来ます。

また、マッチした行を抽出し「行数」と「検索文字」とともにCSVファイル（result.csv）へ出力します。
ログに限らずテキストデータであれば同様に検索とファイルへ出力が可能です。

主にインシデント発生時のログ分析作業を簡略化する手助けが出来ることが目的のプログラムです。

【 ファイル内容 】
strings_search.py   ... 実行、ソースコード
strings_search.exe ... Windows用実行ファイル
　それぞれ単体で実行可能です。

▼ 実行後ファイル
strings_search.log ... 実行結果ログ
result.csv ... マッチした対象の行数と対象行そのものを検索文字に紐づけて出力
→ result.csvの中身
　 検索文字,対象行数,対象行

【 対応するファイルの種類 】
・テキストファイル
・CSVファイル

【 対応OS 】
・Mac
・Linux
・Windows（exeで実行可能）
※いずれもGUI環境があることが前提
